# Диаграмма активности: диаграмма Вороного (с точки зрения пользователя)

Показано, что делает пользователь и что при этом происходит в интерфейсе — без внутренних шагов программы.

```mermaid
flowchart TB
  Start([Пользователь запускает приложение]) --> Wait[Краткое ожидание, индикатор загрузки]
  Wait --> See[Просмотр готовой цветной диаграммы и панели инструментов]

  See --> Choose{Дальше}

  Choose -->|Обновить| Rebuild[Нажать «Обновить»]
  Rebuild --> Building[Короткое ожидание / перерисовка]
  Building --> See

  Choose -->|Сменить число сайтов| Slide[Переместить слайдер и отпустить]
  Slide --> Building

  Choose -->|Открыть ранее сохранённый рисунок| OpenDlg[Нажать «Импорт» и выбрать файл .vjson]
  OpenDlg --> CancelImp{Файл выбран?}
  CancelImp -->|нет| See
  CancelImp -->|да| LoadOk{Загрузка прошла успешно?}
  LoadOk -->|ошибка| MsgErr[Сообщение об ошибке]
  LoadOk -->|успех| MsgOk1[Сообщение, что диаграмма открыта]
  MsgErr --> See
  MsgOk1 --> See

  Choose -->|Сохранить| Export[Нажать «Экспорт»]
  Export --> SaveDlg[Указать имя и место для файла]
  SaveDlg --> CancelExp{Сохранение подтверждено?}
  CancelExp -->|нет| See
  CancelExp -->|да| MsgOk2[Сообщение, что файл сохранён, с путём]
  MsgOk2 --> See

  Choose -->|Просто смотреть| See
```

В файл экспорта попадает описание текущей диаграммы (точки и размер «холста»). При открытии такого файла картинка снова строится в приложении.
